# Credits

## Map node icons

The five map node icons (`crates/helheim/assets/icons/*.png`) are currently
**placeholder shapes** generated for development — distinct white silhouettes
(cross, ring, triangle, square, plus) tinted per node kind at runtime.

They are intended to be replaced with icons from
[game-icons.net](https://game-icons.net) under the
[Creative Commons Attribution 3.0 (CC BY 3.0)](https://creativecommons.org/licenses/by/3.0/)
license. When real icons are added, credit each author here, for example:

- Crossed Swords — Lorc (CC BY 3.0)
- Skull — Lorc (CC BY 3.0)
- Campfire — Delapouite (CC BY 3.0)
- Treasure Chest — Delapouite (CC BY 3.0)
- Crown — Lorc (CC BY 3.0)
